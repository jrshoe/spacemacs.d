## Dependencies:

- Haskell : ghcup -> cabal, stack, ghc (https://develop.spacemacs.org/layers/+lang/haskell/README.html#dependencies)
- Coq : brew -> coqtop
- Racket/Redex : brew -> racket, raco -> drracket, raco -> redex
- Ott : opam -> ott
